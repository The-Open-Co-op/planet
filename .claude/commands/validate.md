Run the build (bun run build) command to validate the code.

**IMPORTANT:** Follow all instructions in CLAUDE.xml throughout this process.

**Workflow**
1. Review CLAUDE.xml for project-specific guidelines and requirements
2. Run `bun run check` to identify typescript and eslint errors
3. Fix all errors found by `bun run check`
4. Repeat steps 2-3 until no errors remain
5. Run `bun run build` to validate that the application builds successfully
6. Fix any build errors if found
7. Repeat steps 5-6 until build succeeds

This command handles the entire validation workflow automatically for non-technical users.
