## Design Principles

1. **Trust as byproduct.** Users never think "I am issuing a credential." The trust graph builds itself through normal behaviour.
2. **Invitation, not marketing.** Every member arrives because someone they trust invited them.
3. **Progressive disclosure.** Day 1 is simple. Depth reveals itself over time.
4. **No trust scores.** Trust is contextual, relational, and multidimensional.
5. **Cooperative by design.** Every member is a potential co-op member.
6. **Public hooks, private depth.** FP Pages drive discovery. Everything else is consent-based.
